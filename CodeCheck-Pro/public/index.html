Key Features

Code Grading

Automatically grade Python and JavaScript code submissions with Docker-based sandboxing for security.

Quiz Grading

Grade quizzes using regex pattern matching with support for CSV upload of answer keys.

Smart Feedback

Provide detailed, rule-based feedback to help students understand their mistakes and improve.

Secure Execution

Run student code in isolated Docker containers to prevent security issues and system damage.
